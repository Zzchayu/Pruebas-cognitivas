# Pruebas Cognitivas — Versión Moderna

Este repositorio contiene una aplicación web en HTML que incluye varias pruebas cognitivas.

## Cómo usarlo
1. Sube estos archivos a un repositorio en GitHub.
2. Activa GitHub Pages desde Settings → Pages.
3. Selecciona la rama `main` y carpeta `/root`.

La página se publicará automáticamente.
